--- layout: post title: 'Bitlash: scriptando na Stellaris e salvando na EEPROM' date: '2014-02-03T23:35:00.002-03:00' author: Murilo Pontes tags: - ARM - prompt de comando - Bitlash modified_time: '2015-01-18T13:11:13.318-03:00' thumbnail: http://1.bp.blogspot.com/-SQXyxx1MTcQ/UvBQLe7nOEI/AAAAAAAAotY/xxjLhdWZWo0/s72-c/bitlash-demo.JPG blogger_id: tag:blogger.com,1999:blog-3047384083282566923.post-8745154054037413442 blogger_orig_url: http://dronespersonalizados.blogspot.com/2014/02/bitlash-scriptando-na-stellaris-e.html --- O Bitlash é uma versão reduzida de um prompt de comandos e controle de tarefas para microcontroladores. Originalmente foi desenvolvido para Arduino / AVR, mas já conta com suporte para alguns kits ARM.
Adicionei o suporte de EEPROM e Software reset para o kit Stellaris Launchpad rodar o bitlash com todos as features disponíveis.

Para instalar o bitlash, basta clonar o repositório dele, em uma pasta sem caracteres especiais dentro da pasta abaixo:
C:\energia-0101E0011\hardware\lm4f\libraries
Feito isso, é só procurar no menu de exemplos da Energia IDE os templates para o bitlash
Dump da eeprom feito pelo comando "peep"
A função "startup" quando definida pelo usuário é chamada sempre a placa é reiniciada.
