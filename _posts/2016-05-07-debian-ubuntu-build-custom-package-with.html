--- layout: post title: 'Debian / Ubuntu: build custom package with checkinstall' date: '2016-05-07T11:27:00.000-03:00' author: Murilo Pontes tags: - Debian - Ubuntu - checkinstall modified_time: '2016-05-07T11:27:20.447-03:00' blogger_id: tag:blogger.com,1999:blog-3047384083282566923.post-4348153167615861621 blogger_orig_url: http://dronespersonalizados.blogspot.com/2016/05/debian-ubuntu-build-custom-package-with.html --- Using checkinstall is easy to manage/remove custom packages.

apt-get -y install checkinstall

wget custom-package.tar.gz
tar xfvz custom-package.tar.gz
cd custom-package
./configure
make
checkinstall make install

# list package contents
dpkg -L custom-package

# remove package
dpkg -r custom-package
