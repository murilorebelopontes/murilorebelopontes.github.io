No TP-Link TL-WR740N dá para usar os GPIO dos leds como I2C com um clock de aproximadamente 83 KHz. 



MPU6050 detectado pelo I2C do TP-Link TL-WR740N
Com o SDK do OpenWRT para TP-Link TL-WR740N é possível criar ferramentas para testar as funcionalidades do MPU6050 e acessar remotamente por WiFi.
